Introduction

This is the modern, jQuery-free version of the PragmaticPlayer. It supports YouTube and HTML5 video APIs with advanced features including keyboard shortcuts, volume control, and playback rate selection.

Features include:

  • Zero dependencies - pure vanilla JavaScript
  • Keyboard shortcuts (Space, M, F, arrow keys)
  • Volume slider with granular control
  • Playback rate selection (0.5x to 2x)
  • Fullscreen support with multiple API fallbacks
  • Accessibility features (ARIA labels)
  • Custom event system
  • Touch-friendly controls

Available Options

youtubeVideoId
The YouTube video ID (required for YouTube adapter)
autoplay
Boolean. Default: false. Auto-start playback
hd
Boolean. Default: true. Enable HD playback for YouTube
origin
String. Your site origin for YouTube API security
adapter
String. Force adapter: "Youtube" or "VideoElement"
features
Array. Default: ['playpause', 'progress', 'timer', 'mute', 'fullscreen', 'volume']. Available: playpause, progress, quality, timer, mute, fullscreen, volume, playbackRate
enableKeyboard
Boolean. Default: true. Enable keyboard shortcuts
